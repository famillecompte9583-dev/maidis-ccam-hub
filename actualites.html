Veille automatique

Actualités et reconstruction des données

La veille est préconstruite côté serveur/GitHub Action pour éviter les blocages CORS côté navigateur.

Pourquoi pas de fetch direct dans le navigateur ?

Les flux RSS et certaines pages institutionnelles bloquent souvent les appels depuis le navigateur. La bonne méthode est de les récupérer côté serveur ou via GitHub Action, puis de reconstruire data/app-data.js. Le site reste ensuite consultable en local.
