Export unitaire

Chaque fiche acte accessible depuis le tableau ouvre un export CSV unitaire pour documenter une vérification ou une correction.

Mapping conseillé

  • Code acte : code CCAM
  • Clé technique : code + activité + phase
  • BRSS : tarif de base
  • Panier : RAC 0 / modéré / libre / hors périmètre
Les exports CSV sont fournis comme aide de travail. Les colonnes peuvent être adaptées selon vos usages et vos règles internes sans remettre en cause le fonctionnement GitHub Pages.
