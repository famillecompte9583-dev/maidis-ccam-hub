Exports et intégration métier

Préparez des fichiers CSV exploitables pour le paramétrage des actes dans Maidis.

Export complet

Depuis la page Actes, cliquez sur “Exporter CSV Maidis”. Le fichier contient code, activité, phase, libellé, BRSS, taux, montant, panier et notes.

Ouvrir la recherche

Export unitaire

Chaque fiche acte possède un bouton d’export unitaire pour documenter un paramétrage ou une correction.

Mapping conseillé

  • Code acte : code CCAM
  • Clé technique : code‑activité‑phase
  • BRSS : tarif de base
  • Panier : RAC 0/modéré/libre/hors périmètre
Les exports CSV fournis sont neutres et documentés. Si votre installation Maidis requiert un format particulier, adaptez simplement les colonnes dans js/actes.js.
