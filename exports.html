Exports

Exporter les résultats

Préparez des fichiers CSV exploitables pour vos vérifications, corrections et travaux de paramétrage.

Export complet

Depuis la page Actes, cliquez sur “Exporter CSV”. Le fichier contient code, activité, phase, libellé, BRSS, taux, montant, panier et notes.

Ouvrir la recherche

Export unitaire

Chaque fiche acte possède un bouton d’export unitaire pour documenter une vérification ou une correction.

Mapping conseillé

  • Code acte : code CCAM
  • Clé technique : code-activité-phase
  • BRSS : tarif de base
  • Panier : RAC 0/modéré/libre/hors périmètre
Les exports CSV sont fournis comme aide de travail. Les colonnes peuvent être adaptées selon vos usages et vos règles internes.
